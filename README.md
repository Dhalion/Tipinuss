<h1 align="center">Tipinuss</h1>
<p align="center"><strong>Online Waschnusswetten</strong></p>

<p align="center">
    Tippe deine Wetten, verdiene Waschnüsse – die moderne Wettplattform für Enthusiasten.
</p>

---

Tipinuss ist eine moderne Wettplattform, gebaut mit **Laravel 12**, **Livewire 4** und **Flux UI**. Dark-Theme-first, responsive, mit vollem Org-Management und Echtzeit-Interaktion.

### Tech-Stack

- **Backend:** Laravel 12, PHP 8.4
- **Frontend:** Livewire 4, Flux UI, Alpine.js, Tailwind CSS v4
- **Database:** SQLite / MySQL (via Laravel)
- **Build:** Vite 7
